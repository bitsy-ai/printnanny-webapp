About

Print Nanny's AI uses single-shot multibox detector (a type of neural network) to track various aspects of your print.

This computer vision technique can detect multiple "classes" (or "labels") in one pass over image data, which makes it ideal for real-time applications like autonomous driving, smart security cameras, and quality assurance systems like Print Nanny.
